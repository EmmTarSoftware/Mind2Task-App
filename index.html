    Clôture de session

    En cliquant sur le bouton "Clôture de session", vous confirmez votre intention de terminer votre session actuelle. Voici ce qui se passera lors de cette opération :

    1. Extraction de la base de données "Statistiques" : La base de données des statistiques de l'application sera extraite depuis la tâche la plus ancienne jusqu'à la plus récente et sera sauvegardée pour référence future.
    2. Réinitialisation de la base de données "Statistiques": Après l'extraction, la base de données des statistiques sera réinitialisée. Cela entraînera la suppression de toutes les statistiques, les remettant à leur état initiale. Cependant, vos informations extraites seront toujours disponibles pour consultation ultérieure.

    Veuillez noter que cette action est irréversible et qu'elle effacera toutes les statistiques de l'application.
